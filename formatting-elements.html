Text Formatting

  • Use headings 'h1' through 'h6' to define the hierarchical structure of your content, with 'h1' being the most important and 'h6' being the least.
  • Paragraphs are created using the 'p' element
  • Other formatting elements include 'strong' for bold text, 'em' for italic text, 'u' for underlined text (although not recommended for modern web design due to accessibility concerns), and 's' for strikethrough text
